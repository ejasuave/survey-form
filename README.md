# Homemade Food Delivery Preference Survey

A tailored, high-conversion customer research form built by **Suaviiz** for an upcoming homemade food delivery service. This form was engineered to collect crucial user preferences on dietary habits, meal choices, and delivery scheduling to guide product launch decisions.

The layout uses a high-contrast, modern corporate-creative palette carefully selected to drive form completion rates:

---

# Architectural Breakdown (7 Specialized Sections)

The form uses logical sequencing across 7 core structural data points to capture specific consumer habits for the client:
1. **User Info:** Collecting baseline demographics (Name, Age, and Email).
2. **Online Ordering Frequency:** Measuring how often the user orders food online.
3. **Prior Homemade Experience:** Discovering if the user has ever ordered homemade food online before.
4. **Cuisine Preference Selection:** Identifying what types of homemade meals the user fancies most.
5. **Delivery Speed Valuation:** Quantifying how important fast delivery is to the user.
6. **Ideal Scheduling:** Mapping out the user's ideal delivery time for homemade food.
7. **Menu Variety Importance:** Grading how important it is to have a wide variety of homemade meals available at any time.g

# Technical Implementation & Challenges Overcome

* **The Challenge:** Achieving perfect structural alignment between custom checkboxes/radio selections and their descriptive text nodes.
* **The Solution:** Implemented modern **CSS Flexbox alignments** and specific input padding constraints to ensure text elements stay perfectly inline across desktop and varying mobile display dimensions.
* **Markup:** 100% semantic HTML5 inputs, utilizing proper `<label>` associations for elevated accessibility.
